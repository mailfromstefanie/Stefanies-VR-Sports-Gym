# Stefanie's VR Sports Gym

VRChat sports hall with multiple ball-game modes, shared scoreboards and Quest/PC support. Built with Unity 2022.3 LTS and UdonSharp.

## Start here

For a new development chat, use:

1. `START_PROMPT.md`
2. `PROJECT_IDENTITY.md`
3. `CURRENT_WORK.md`
4. `SOURCE_OF_TRUTH_RULES.md`
5. `COLLABORATION_RULES.md`
6. the active feature folder named in `CURRENT_WORK.md`

The mobile copy-and-paste start prompt is available at:

`FOR_STEF/Startprompt_Stefanies_VR_Sports_Gym.txt`

## Current project truth

`CURRENT_WORK.md` determines:

- the active feature;
- the current phase and microstep;
- what has already been completed;
- what may be changed now;
- what must not be changed yet;
- the pass condition and next step.

Reading the repository is not permission to jump ahead. Work one small, verified step at a time and wait for Stef's real test result before continuing.
